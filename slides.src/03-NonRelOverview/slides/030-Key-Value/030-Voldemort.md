## Project Voldemort

Факты

* 2009, LinkedIn
* AP в терминах CAP

Свойства

* Автоматическая репликация
* Автоматическое разделение
* Механизмы для сериализации сложных ключей и значений (но не в «ядре», плагины)
* Механизмы для использования разных сисиетм хранения
* Версионирование
* Оптимизирована для SSD
